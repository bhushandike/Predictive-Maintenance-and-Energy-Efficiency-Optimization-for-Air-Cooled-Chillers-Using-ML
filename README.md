# Predictive Maintenance and Energy Efficiency Optimization for Air Cooled Chillers Using ML

## Project Overview
This project is focused on developing a machine learning framework for predicting maintenance needs and optimizing energy efficiency in air-cooled chillers. By utilizing historical operational data, our system aims to anticipate failures and reduce energy consumption, leading to significant cost savings and improved reliability.

## Features
- Predictive maintenance using machine learning algorithms.
- Energy consumption optimization strategies.
- Easy-to-use dashboard for monitoring and analytics.
- Comprehensive reports on predictive analytics and energy efficiency.

## Installation
### Prerequisites
- Python 3.7 or newer
- Pip package manager

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/bhushandike/Predictive-Maintenance-and-Energy-Efficiency-Optimization-for-Air-Cooled-Chillers-Using-ML.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Predictive-Maintenance-and-Energy-Efficiency-Optimization-for-Air-Cooled-Chillers-Using-ML
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
- After installation, run the application with:
   ```bash
   python app.py
   ```
- Follow the on-screen instructions for data input and analysis.

## Technical Details
This project employs various machine learning models, including:
- Linear Regression
- Decision Trees
- Random Forest
- Support Vector Machines (SVM)

Data preprocessing and feature engineering techniques are extensively used to improve the performance of the models used in this project. The models are trained and validated using historical data provided in the `data/` directory.
